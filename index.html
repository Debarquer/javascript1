How Bish Bosh Works

Bish Bosh is a simple game where you count from 1 to a given number. For each number, if it is divisible by a specified "Bish" number, you say "Bish". If it is divisible by a specified "Bosh" number, you say "Bosh". If it is divisible by both, you say "Bish Bosh". Otherwise, you just say the number.

The purpose of this page is to let you try out this game in an accessible manner.

Try out different values for the length, Bish and Bosh to see how the game changes.
